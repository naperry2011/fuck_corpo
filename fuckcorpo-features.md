# FuckCorpo - Feature List

## Core Concept
A web app for tracking and calculating money earned while taking bathroom breaks on company time.

---

## Core Functionality Enhancements

### Timer Modes
- **Quick log** - Manual entry for past breaks
- **Live timer** - Real-time tracking during break
- **Automatic tracking** - Set it and forget it

### Salary Calculator
- Convert annual salary to per-minute rate automatically
- Support for hourly, salary, and contract workers

### Break Categories
- Bathroom breaks
- Smoke breaks
- "Mental health moments"
- Coffee breaks
- Other custom categories

### Running Totals
- Daily earnings
- Weekly earnings
- Monthly earnings
- Yearly earnings
- Lifetime total

## Technical Considerations

### Platform
- **Mobile-first design** (people use phones in bathrooms)
- **PWA (Progressive Web App)** for easy access without app store
- Responsive across all devices
- Fast loading times

### UX/UI
- **Dark mode** (essential for bathroom lighting)
- **Light mode** (for the brave)
- Large, easy-to-tap buttons
- Simple, intuitive interface
- Quick start/stop functionality

### Performance
- Offline capability
- Local storage fallback
- Fast calculations
- Minimal data usage
